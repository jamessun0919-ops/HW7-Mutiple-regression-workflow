# California Housing Feature Selection & Stepwise Evaluation

This project implements an automated, general-purpose machine learning pipeline to perform feature selection and stepwise model evaluation on any tabular regression dataset, as described in `workflow.md`. 

Here, the workflow is applied to the **California Housing** dataset.

## 📈 Analysis Results

Below is the publication-quality dual-curve and ranking table visualization generated by the pipeline:

![California Housing Feature Selection Performance](feature_selection_performance_allinone.png)

### Key Insights:

1. **Top Feature**:
   - `median_income` is consistently selected as the **Rank 1 feature** across all 9 selection algorithms (Pearson, Spearman, F-test, Mutual Info, RFE, SFS, SBS, Lasso L1, and Random Forest).
   - This validates the hypothesis that median area income is the primary driver of house value.

2. **Subset Sweet Spot (Elbow Point)**:
   - According to the **Test MSE** curve, the elbow point occurs at **$k = 4$ or $k = 5$ features**.
   - Adding features beyond 5 yields minimal performance improvements while increasing model complexity.
   - Using only the top 5 features achieves a Test R-squared of **`0.61`**, which is very close to the full-feature performance of **`0.614`** (with 12 features after encoding).

## 🛠️ Step-by-Step Instructions

### 1. Requirements
Ensure you have the required packages installed:
```bash
pip install pandas numpy scikit-learn matplotlib scipy
```

### 2. Run the Pipeline
To run the automated analysis:
1. Ensure your dataset is named `data.csv` in the root folder.
2. Execute the runner script:
   ```bash
   python run_workflow.py
   ```
3. The script will automatically:
   - Detect target and feature types (handles numeric standardization and categorical one-hot encoding).
   - Compute rankings using 9 distinct feature selection methods.
   - Run stepwise linear regression for $k = 1$ to $N$ features.
   - Save the visualization as `feature_selection_performance_allinone.png`.
